# 🚀 Niagara

Niagara is a Fabric mod library for Minecraft that allows mod developers to create HTML/CSS/JS powered UIs using [Ultralight](https://ultralig.ht/).

To whoever on Solomon Team it may concern, this is basically a significantly better version of [Luminescent](https://github.com/Ayydxn/Luminescent). Please do not continue using that garbage.
